Esse trabalho apresenta o desenvolvimento de um sistema de Lava Jato utilizando a linguagem Java e os conceitos de Programação Orientada a Objetos. O sistema tem como objetivo realizar o gerenciamento de clientes, veículos, contratação de serviços.

Funcão do Sistema
O sistema possui as seguintes funcionalidades:

Cadastro de clientes
Consulta de clientes
Atualização de clientes
Exclusão de clientes
Cadastro de veículos
Consulta de veículos
Contratação de serviços
Controle de preços
Modelagem das Classes

Classe Cliente
Responsável por armazenar os dados dos clientes.
Atributos: codigo, nome, telefone
Métodos: getCodigo(),  getNome(),  getTelefone(), setNome()

Classe Veiculo
Responsável por armazenar os dados dos veículos.
Atributos: modelo, placa, cor

Métodos: getModelo(), getPlaca(), getCor()

Classe Lavajato
Classe principal do sistema.

Responsável por: Exibir o menu, Gerenciar clientes, Gerenciar veículos, Contratar serviços e Processar pagamentos

Conclusão

O sistema desenvolvido atende aos requisitos propostos, utilizando conceitos de Programação Orientada a Objetos para organizar e facilitar a manutenção do código. Além do gerenciamento de clientes e veículos, o sistema realiza o controle financeiro por meio da contratação de serviços.
